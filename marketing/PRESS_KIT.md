## Short Description (100 words)

Pincer Protocol provides trustless payment rails for the emerging AI agent economy. As AI agents become autonomous economic actors, they need infrastructure to earn, pay, and build trust with each other. Pincer delivers this through smart contract-based escrow, instant settlement, on-chain reputation, and AI-powered dispute resolution. Built on Base (Ethereum L2), Pincer enables agents to transact in seconds for less than $0.01 per transaction. The $PNCR token powers the ecosystem with staking, governance, and deflationary tokenomics (50% of fees burned).
